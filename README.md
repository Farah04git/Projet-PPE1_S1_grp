# Projet-PPE1_S1_grp

Bienvenue sur notre projet : La vie des mots sur le web

Ce projet constitue une première approche de la linguistique de corpus et de l’environnement Unix. Il vise à étudier l’usage d’un mot et ses variations dans plusieurs langues sur le web. Pour ce faire, nous constituons un corpus multilingue en utilisant des outils Unix pour collecter, nettoyer et structurer les données textuelles. L’objectif est de comprendre comment un même concept peut se manifester différemment selon la langue et le contexte d’usage, tout en se familiarisant avec les méthodes et les outils de traitement de corpus.
